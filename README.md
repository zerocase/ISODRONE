# ISODRONE

ISODRONE is a **polyphonic vocal drone synthesizer plugin** built with [JUCE](https://juce.com/).  
It is designed for creating evolving ambient textures and soundscapes, featuring an isomorphic MIDI input processor and a clean, customizable interface.

---

## Features
- Polyphonic vocal drone synthesis engine  
- Custom isomorphic MIDI mapping  
- Cross-platform support (VST3, AU, Standalone)  
- GUI built with JUCE  
- Open-source and extensible  

---

### Building from Source
1. Install [JUCE](https://juce.com/get-juce).  
2. Open `ISODRONE.jucer` in the **Projucer**.  
3. Select your target exporter (Visual Studio, Xcode, Makefile, etc.).  
4. Save and open the generated project in your IDE.  
5. Build the project to generate the plugin and standalone app.  

---

## Usage
1. Load **ISODRONE** in your DAW.  
2. Connect a MIDI keyboard or sequencer.  
3. Play notes to generate drones and textures.  
4. Use the GUI controls to shape the sound in real time.
